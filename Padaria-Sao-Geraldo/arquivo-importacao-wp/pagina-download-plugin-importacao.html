Descrição

Este plugin exporta o seu site em WordPress incluindo o banco de dados, arquivos de mídia, plugins e temas, sem necessidade de qualquer conhecimento técnico.
Envie o seu site para um local diferente arrastando e soltando no seu WordPress.
Há uma opção de aplicar um número ilimitado de operações de encontrar e substituir no seu banco de dados durante o processo de exportação. O plugin também irá corrigir qualquer
problema de serialização que possa ocorrer durante a operação de encontrar/substituir.

Compatível com dispositivos móveis: o All in One WP é o primeiro plugin a oferecer a verdadeira experiência em dispositivos móveis desde o WordPress 3.3 e acima.

Nenhuma limitação de sistema operacional na hospedagem

  • Testamos o plugin nas principais distrubuições do Linux, no macOS e no Microsoft Windows.
  • Veja a lista de provedores de hospedagem com os quais nós funcionamos.

Contorne todas as restrições de tamanho de envio

  • Usamos pedaços para importar os dados do seu site. A maioria dos provedores define um tamanho máximo de envio de 2MB. Como as restrições de arquivo são aplciadas apenas a cada pedaço, as restrições de tamanho de envio do servidor são contornadas ao mantermos pedaços menores do que 2MB para enviar facilmente o seu site completo.

Zero dependências

  • O plugin não requer quaisquer extensões do PHP e funciona com todas as versões do PHP desde a v5.2 em diante. Isso é uma excelente notícia para os usuários da v5.2 que não têm o suporte de muitos outros produtos.

Suporte a MySQL e MySQLi

  • Não importa qual driver MySQL para PHP seu servidor usa, nós o suportamos.

Compatível com o WordPress v3.3 até a atual

  • Temos um processo extenso de Garantia de qualidade e testes, que garante que o plugin é sempre compatível com a última versão do WordPress, mas não damos suporte a versões do WordPress anteriores à versão 3.3 (de 2012)

A integração ao WP-CLI está disponível na Extensão Ilimitada

Instalação

  1. O All-in-One WP Migration pode ser instalado diretamente do WordPress
    Plugins no painel.
  2. Clique em “Adicionar novo” e procure por “All-in-One WP Migration”
  3. Instalar e Ativar

Alternativamente você pode baixar o plugin usando o botão de baixar nesta página, então enviar a pasta all-in-one-wp-migration para o diretório /wp-content/plugins/ e ativá-lo através da página de Plugins do painel do WordPress
